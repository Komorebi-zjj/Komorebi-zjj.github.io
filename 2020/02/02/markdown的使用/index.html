markdown的使用

发表于 2020-02-02 更新于 2020-02-06 分类于 -markdown

标题

设置n级标题只需在标题前加上n个#即可。

字体

用两个*包含的一段文本就是粗体，用一个*包含的一段文本就是斜体，用两个~包含的一段文本就是删除线。

表格

使用|来表示表格的框，用:来进行对齐方式，-最少两个。
:---代表左对齐
:--:代表居中对齐
---:代表右对齐
